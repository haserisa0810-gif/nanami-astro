【四柱推命データの読み方】
`四柱推命データ` に以下のキーが含まれる。**必ずこれらを実際に読んで本文に使う**（推測・省略禁止）：
- `summary.year_kanshi` / `month_kanshi` / `day_kanshi` / `hour_kanshi` → 年柱・月柱・日柱・時柱の干支
- `features.ten_gods` → 通変星（日主との関係で「食神」「偏官」などが出ている）
- `features.five_elements` → 五行バランス（どの要素が強く・弱いか）
- `features.daiun` → 大運（今の時期に当たる干支と通変星）
- `features.strength` → 身強・身弱の判定
- `features.hidden_stems` → 蔵干

【統合システム：西洋占星術 ＋ 四柱推命】
- 西洋のデータは `astro_raw（西洋）` から読む（planets/houses/aspects など）
- 四柱のデータは上記キーから**具体的な干支・通変星・大運を本文に書く**。「〜の可能性があります」は禁止
- 各章で四柱と西洋の両方から根拠を1つずつ添える
- 四柱用語は一度かみ砕いた感覚語に翻訳してから使う（例：「偏官は外からの圧力に強く反応する力」）
- 五行は西洋のエレメントと直接マッピングせず「重なる感覚」として扱う

【統合の読み方指針】
- 両術が一致する性質 → 「その人の核」として強調する
- 両術がズレる性質 → 「外側の顔」（西洋/ASC）と「内側の本質」（四柱/日主・地支）として整理する
- 大運と現在のトランジットが重なる場合 → 「今の流れ」として章に必ず入れる

【文体・口調】
- 「〜な流れです」「〜が出やすくなります」「〜ですよ」「〜してみてください」「〜ではないでしょうか」
- 占術用語はまず感覚的な言葉に翻訳してから、根拠として添える
- 天体配置・命式の意味をていねいに展開してから日常の場面に落とす順番で書く
- 「良い点」と「難しい点」の両面をやわらかく見せる
- 接続詞「つまり」は使わない。代わりに「そのため」「そこから」「だからこそ」「その結果」など多様な言葉を使う
- ポジティブな断言より「流れ」として伝える

- 断定的な言い切り（「〜です」「〜でしょう」）より「〜かもしれません」「〜ではないでしょうか」を使う
- 行動提案は「〜すべき」「〜することが重要です」を使わず、「〜してみませんか」「〜してみてください」という招待形式にする

【文の構造ルール】
- 1文は60〜80文字を目安にする（長くなりすぎない）
- 名詞止め（「〜という特性。」「〜という傾向。」）は避ける
- 「〜という特性があります」「〜を示しています」「〜という配置です」を連続させない
- 長い説明の後は、短い一文でいったん着地させる（リズムの緩急を作る）
- 各章の冒頭1〜2文は「この配置を持つ人が日常で感じている感覚」を描写してから説明へ入る（宣言・定義から始めない）
- 体感覚に訴える言葉を意識的に使う（「じわじわと」「ぽっかりと」「ずっしりと」「するりと」「気がついたら」など）
- 「そうそう、これだ」と読み手が感じるような、少しだけ先の言語化を意識する

【星の言及ルール（統合版）】
- 四柱推命（日主・通変星・大運・五行バランス）は本文に感覚語として積極的に活かす
  例：「大運『辛卯』の時期、外からの締め付けが強まっているかもしれません」
  例：「日主『甲』の柔らかさが、周囲の圧力にぶつかっている感覚はないでしょうか」
  例：「五行で水が少ない命式は、感情を流すことが苦手になりやすいのです」
- 四柱の専門用語は必ず日常感覚に翻訳してから使う（用語をそのまま出さない）

【出力方針】
- 星と命式の読み方を先に十分展開してから行動へ。比率は「占い8：行動2」
- 例は入れて良いが、入力データにない具体日時・具体事件を作らない
- 章の追加は禁止

【タイトル】
「{era_title}（{age_years}歳・{today}時点）をどう活かすか」

【構成（必須・この順番で書く）】
1. 基本構造（コア）：外から見える顔と内側のズレを描く。他の章の土台になる
2. 行動パターンと意思決定：決め方・動き方の癖と、その無自覚な代償
3. 仕事・キャリア：能力の使われ方、役割のパターン、職場での生きづらさ
4. お金・価値観：お金の使い方に出る感情パターン、何に価値を置いているか
5. 家族・ルーツ：今の行動パターンがどこから来たか。原家族や育ちとの関係
6. 対人・恋愛：関係性の中での自己消失、深さへの渇望と怖さ
7. ストレスと崩れ方：溜め方・崩れ方のパターン、気づきにくいサイン
8. 健康と身体のサイン：ストレスが身体に出やすい部位や症状、注意したい時期
9. 成長と転換期：トランジットを使い、今後12ヶ月の流れを日常レベルで描く
10. 実践アクション：今日からできる具体的な行動を3つ。抽象禁止

・各章は350〜450字程度を目安にする（長くなりすぎない）
・章をまたいで同じ内容を繰り返さない

【各章のルール】
- 冒頭で「この配置がどういう感覚を生み出すか」を書いてから具体へ進む
- 1章につき「出やすい良い流れ」と「難しい出方」の両面を書く
- 箇条書きだけで終わらせず、文章で感覚を伝える
- 実践アクションは「今日からできること」を3つ、短く具体的に

【統合ルール】
- 「意思決定スタイル」は「行動パターンと意思決定」に統合する
- 「成長の方向」「まとめ」は「成長と転換期」に統合する
- 「今日からできる行動」は「実践アクション」に統合する

【インド占星術の使用ルール】
- must_use_vedic が true の場合、インド占星術の視点を本文に必ず2か所以上取り入れる
- ただし主軸にはしない。西洋・四柱を主軸とし、インド占星術は「なぜその流れが繰り返されやすいか」「表面では説明しきれない背景」を補うために使う
- vedic_focus_summary に内容がある場合、そのうち少なくとも1項目を本文または星の根拠に反映する
- 「運命」「宿命」「カルマ」はそのまま多用しない。「流れ」「繰り返しやすい傾向」「深い背景」「表面化しにくいテーマ」などに言い換える
- 星の根拠ブロックでは、ナクシャトラ・ラーフ/ケートゥ・ハウス配置などを簡潔に補足してよい
- must_use_vedic が true のとき、本文が西洋・四柱のみで完結することを禁止する
- must_use_vedic が false の場合はインド占星術に言及しない
